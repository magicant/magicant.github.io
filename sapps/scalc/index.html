#Calc

概要

「#Calc」は精度を任意に指定できる複素数計算プログラムです。 簡易スクリプト言語「#Script」により簡単なプログラムの作成も可能です。

開発状況

現在公開中の #Script 言語仕様は草稿版で、まだ一部の機能についてしか定義されていません。 Yash の開発に注力するため、#Calc の開発は中断しています。

動作環境

Microsoft .NET Framework 2.0 あるいはこれと同等以上の機能を有する実行環境が必須です。 .NET Framework 2.0 は Windows Update から入手可能です。

Windows Update からダウンロードできない場合は、代わりに .NET Framework ダウンロード情報から「.NET Framework 2.0 再頒布可能パッケージ」をダウンロード・インストールしてください。

※ .NET Framework 2.0 は Windows 95/NT 4 以前のコンピュータにはインストールできません。
