#Calc について

#Calc (シャープ カルク) は計算の精度を自由に設定できる複素数計算ソフトです。 独自の簡易プログラミング言語 #Script (シャープ スクリプト) のインタプリタとして動作するので、簡単なプログラム電卓として使用することもできます。

動作環境

#Calc を動作させるには、Microsoft .NET Framework 2.0 あるいは これと同等以上の機能を有する実行環境が必須です。 .NET Framework 2.0 は Windows Update から入手可能です。

インストールとアンインストール

#Calc をインストールするには、新しくフォルダを作ってそこにアーカイブの内容を全て コピーしてください。インストーラ (インストールを行うための特別なプログラム) はありません。

#Calc が不要になりましたら、そのまま削除してください。 (現在のバージョンでは設定の保存などは一切行っておりません)
