トラックバック ping 送信

© 2006 Magicant / 0.0 (2006-07-27)

第三者トラックバックを送信するためのフォームです。任意のページから任意のページにトラックバック ping を送信できます。

このフォームはDrifting Antigone Frontline » 外部トラックバック続報　投げ銭にあるものを改造したものです。

トラックバック元に関する情報
ページの題名 (タイトル)
ページの内容の要約
ページの URI (必須)
サイトの名前
トラックバック先に関する情報
トラックバック ping 送信先 URI (必須)

使い方

上のフォームの各項目に記入し、送信ボタンを押してください。

Ping の送信が正常に終了すると、次のような XML 文書が表示されます:

<?xml version="1.0" encoding="iso-8859-1"?>
<response>
<error>0</error>
</response>

お使いのブラウザによっては、XML 文書の内容が解釈されて単に 0 などと表示されるかもしれません。

Ping の送信が失敗した場合は、次のような XML 文書が表示されます:

<?xml version="1.0" encoding="iso-8859-1"?>
<response>
<error>1</error>
<message>The error message</message>
</response>

使用上の注意

一度送信したトラックバック ping を取り消すことはできません。送信する前に入力した内容を十分良く確認してください。 とくに、トラックバック ping 送信先 URI は間違えないようにしてください。
